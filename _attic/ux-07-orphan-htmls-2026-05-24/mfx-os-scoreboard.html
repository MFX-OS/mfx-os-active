5 Target Workflows
Coverage of each workflow against what's built in MFX OS today.
1. Quote-to-Cash
50%
Client specs → Quote created (material calculator exists)
Pricing is formula-based, not AI-driven from specs input
CEO approval flow → Quote status updates
Quote → Sales Order conversion
No client self-service portal for approve & pay
No QuickBooks auto-invoice generation
2. File-to-Press
45%
Client uploads PDF → files stored in Drive per job
No AI pre-flight (bleed, resolution, color mode checks)
Proof tracking + approval records in PPD module
Job Ticket generated with prepress stage tracking
No digital push to press operator iPad / shop floor
3. Roll-to-Pouch
10%
Production queue exists (prepress → running → QC → done)
No raw material loading / roll tracking
No print run tracking (footage, speed, waste)
No lamination / slitting / converting stages
No finished goods inventory or pallet tracking
4. Inspect-to-Correct
5%
PPD plate incident log captures defect records manually
No inline camera / vision system integration
No real-time defect detection
No automatic press adjustment or halt
5. Sensor-to-Service
0%
No IoT sensor integration
No machine health monitoring
No predictive maintenance AI
No auto-scheduled maintenance tickets
